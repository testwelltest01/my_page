# 09. 리액트에서 아이콘(Icon) 사용하기

웹사이트를 풍성하게 만드는 아이콘들을 어떻게 가져오고 사용하는지 알아봅니다. 특히 **`import`** 방식을 활용한 현대적인 아이콘 라이브러리 사용법을 중점적으로 다룹니다.

---

## 1. 아이콘을 가져오는 두 가지 방식

### ① 전통적인 방식 (이미지)
*   **방법**: `.svg`나 `.png` 파일을 저장한 뒤 `<img>` 태그로 불러오기.
*   **단점**: 크기 조절이나 색상 변경이 어렵고, 파일이 많아지면 관리가 힘듭니다.

### ② 현대적인 방식 (컴포넌트로 import)
*   **방법**: 아이콘 전용 라이브러리를 설치한 뒤, 자바스크립트 함수(컴포넌트)처럼 **`import`** 해서 사용.
*   **장점**: **글자(Text)처럼 다룰 수 있습니다.** 크기(`size`)와 색상(`color`)을 코드 한 줄로 쉽게 바꿀 수 있습니다.

---

## 2. 가장 많이 쓰이는 라이브러리: Lucide React & React Icons

우리가 `02번` 파일에서 배웠던 **`npm install`**을 사용할 차례입니다!

### 🌟 Lucide React (깔끔하고 세련된 아이콘)
요즘 Vite나 Next.js 프로젝트에서 가장 유행하는 라이브러리입니다.

*   **설치**: `npm install lucide-react`
*   **사용법**:
```jsx
// 1. 사용할 아이콘만 콕 집어서 import 하기
import { Home, User, Settings, Heart } from 'lucide-react';

function App() {
  return (
    <div className="flex gap-4 p-10">
      {/* 2. 컴포넌트처럼 사용하고, 속성(Props)으로 디자인 조절 */}
      <Home color="blue" size={32} />
      <User color="red" size={32} />
      <Settings className="animate-spin" /> {/* Tailwind로 애니메이션도 가능! */}
      <Heart fill="red" color="red" />
    </div>
  );
}
```

---

## 3. 아이콘을 왜 컴포넌트로 import 하나요?

1.  **용량 최적화 (Tree Shaking)**: 수천 개의 아이콘 중 내가 **`import`** 한 코드만 딱 골라서 빌드합니다. (나머지는 버려지므로 웹사이트가 가벼워집니다.)
2.  **Tailwind CSS와의 완벽한 조화**: 
```jsx
<Search className="w-6 h-6 text-gray-400 hover:text-blue-500 transition" />
```
위와 같이 아이콘에 직접 테일윈드 클래스를 붙여서 스타일을 입힐 수 있습니다.

---

## 4. 아이콘 찾기 꿀팁

직접 그릴 필요 없이 아래 사이트들에서 이름을 검색한 뒤, 그 이름 그대로 `import` 하면 됩니다.

*   **[Lucide.dev](https://lucide.dev/icons)**: 심플하고 모던한 아이콘 (강력 추천)
*   **[React Icons](https://react-icons.github.io/react-icons/)**: FontAwesome, Material Design 등 거의 모든 유명 아이콘 세트를 하나로 모아둔 백화점 컨셉.

---

## 💡 요약
리액트에서 아이콘은 "파일"이 아니라 **"코드(컴포넌트)"**입니다.

1.  **`npm install`**로 아이콘 뭉치를 가져온다.
2.  내가 쓰고 싶은 아이콘의 이름을 **`import { IconName }`**으로 부른다.
3.  **`<IconName />`** 태그를 써서 화면에 내보낸다.

이제 아이콘 하나로 훨씬 더 프로페셔널한 UI를 만드실 수 있습니다! 😊
