# 📂 분야별 주요 개발 도구 정리

이 문서는 개발 분야별로 프로젝트를 시작할 때 쓰는 **뼈대 도구(Create)**와 기능을 개발할 때 쓰는 **재료 라이브러리(Install)**를 정리한 가이드입니다.

---

## 1. 🌐 웹 프론트엔드 (Web Frontend)
웹사이트의 화면과 사용자 경험을 담당하는 분야입니다.

### 🏗️ 프로젝트 시작 (Create)
- **Vite**: `npm create vite@latest`
  - 가장 빠르고 현대적인 번들러입니다. React, Vue, Svelte 등을 지원합니다.
- **Next.js**: `npx create-next-app@latest`
  - 서버 사이드 렌더링(SSR)이 강점인 React 프레임워크입니다. SEO에 필수적입니다.

### 🚀 서버 실행 방법 (Run Server)
- **Vite**: `npm run dev`
- **Next.js**: `npm run dev`

### 🛠️ 필수 기능 추가 (Install)
- **Axios**: `npm install axios`
  - HTTP 통신 라이브러리입니다. 백엔드 API에서 데이터를 가져올 때 사용합니다.
- **Tailwind CSS**: `npm install -D tailwindcss`
  - 유틸리티 우선 CSS 프레임워크로, 디자인 작업을 코드 내에서 빠르게 처리합니다.
- **Zustand**: `npm install zustand`
  - 가볍고 직관적인 상태 관리 라이브러리입니다. (로그인 정보, 장바구니 등 관리)

---

## 2. 📱 모바일 앱 (Mobile App)
아이폰과 안드로이드에서 실행되는 애플리케이션 개발 분야입니다.

### 🏗️ 프로젝트 시작 (Create)
- **Expo**: `npx create-expo-app [프로젝트명]`
  - React Native를 가장 쉽게 시작할 수 있는 도구입니다. 복잡한 네이티브 설정 없이 바로 시작 가능합니다.
- **React Native CLI**: `npx react-native init [프로젝트명]`
  - 조금 더 깊은 하드웨어 제어가 필요할 때 사용하는 정석적인 방법입니다.

### 🚀 서버 실행 방법 (Run Server)
- **Expo**: `npx expo start` (또는 `npm start`)
- **RN CLI**: `npm run android` 또는 `npm run ios`

### 🛠️ 필수 기능 추가 (Install)
- **React Navigation**: `npm install @react-navigation/native`
  - 앱 내 화면 이동(네비게이션)을 구현하는 핵심 도구입니다.
- **Lottie**: `npm install lottie-react-native`
  - 에어비앤비에서 만든 애니메이션 라이브러리로, 앱에 세련된 애니메이션을 추가합니다.

---

## 3. ⚙️ 백엔드 (Backend / Server)
데이터 저장, 가공 및 서버 로직을 담당하는 분야입니다.

### 🏗️ 프로젝트 시작 (Create)
- **NestJS**: `npx @nestjs/cli new [프로젝트명]`
  - 효율적이고 유지보수가 쉬운 대규모 서버를 구축할 때 사용하는 전문적인 프레임워크입니다.
- **Express Generator**: `npx express-generator`
  - 가장 대중적인 Node.js 웹 프레임워크인 Express의 기본 틀을 잡아줍니다.

### 🚀 서버 실행 방법 (Run Server)
- **NestJS**: `npm run start:dev` (수정 시 자동 재시작)
- **Express**: `npm start`

### 🛠️ 필수 기능 추가 (Install)
- **Prisma**: `npm install prisma`
  - 데이터베이스를 자바스크립트 객체처럼 쉽게 다루게 해주는 ORM 도구입니다.
- **MySQL2 / MongoDB**: `npm install mysql2` / `npm install mongoose`
  - 실제 데이터베이스 서버와 연결해주는 드라이버/라이브러리입니다.
- **Dotenv**: `npm install dotenv`
  - API 키나 DB 비밀번호 같은 민감한 정보를 안전하게 관리하는 설정 도구입니다.

## 💡 한눈에 정리
| 구분 | 뼈대 도구 (Create) | 설치/재료 (Install) |
| :--- | :--- | :--- |
| **목적** | 프로젝트 폴더 및 기초 설계 생성 | 특정 기능(통신, 디자인, DB) 추가 |
| **특징** | `npx`로 실행 후 삭제되는 경우가 많음 | `package.json`에 영구 기록됨 |
| **비유** | 집의 설계도와 골조 세우기 | 가구, 벽지, 가전제품 들여놓기 |
